# micro-soc-tickets (Version 17)

Esta es una pagina web que simula las peticiones de tickets y la incorporacion de un un chatbot para la asistencia del cliente.
Las paginas cuentan con HTML, CSS y Javascripts de por medio, para su funcionamiento actual, los datos ingresados se guardan en una base de datos alojada en Supabase.

# Funciones de micro-soc-tickets
- Sistema de tickets para soporte
- Preguntas frecuentes añadidas
- Un chatbot aplicado que atiende las 24 horas del dia.
